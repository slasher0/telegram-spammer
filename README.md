# telegram-spammer
Telegram spammer + report bot + botnet с поддержкой импорта аккаунтов с lzt market по Auth Key (HEX)

Мощный и функциональный локальный инструмент для управления сессиями Telegram, рассылки сообщений и массовых жалоб (репортов).

## 🚀 Функционал
- **Account Warehouse:** Импорт сессий из различных форматов (.tdata, .session, raw keys) с автоматической конвертацией в легкий формат `.string`.
- **Proxy Manager:** Умное управление прокси (SOCKS5), автоматическая привязка прокси к аккаунтам, проверка на работоспособность.
  - *Auto-Distribute:* Автоматическое распределение списка прокси по всем загруженным аккаунтам.
  - *Redistribute Failed:* Переназначение прокси для аккаунтов, которые выдали ошибку при последней проверке.
  - *Scrape Proxies:* Встроенный парсер для сбора бесплатных прокси прямо в приложении.
- **Attack Panel:** Мощный движок для агрессивной рассылки сообщений и медиа-файлов по заданным целям. Встроенная обфускация текста.
- **Report Panel:** Автоматизированная система отправки жалоб (Spam) на выбранный аккаунт или канал.
- **Global Logs:** Детальное логирование всех процессов в реальном времени.

- Установите необходимые зависимости:

pip install -r requirements.txt
```

Запустите приложение:

python gui_app.py
```

## 📁 Структура проекта
- `gui_app.py` — Главный файл с графическим интерфейсом (GUI).
- `core_engine.py` — Основная логика: управление сессиями, движок рассылки и репортов.
- `proxy_manager.py` — Модуль для работы с прокси-серверами.
- `account_importer.py` — Модуль импорта и конвертации аккаунтов.
- `telegram_session_encoder.py` — Внутренний кодировщик для создания StringSession из сырых ключей.
